## Installation
Teeko requires the python3-pil.imagetk package and Pillow

```bash
git clone https://github.com/mlcsthor/teeko.git

apt-get install python3-pil.imagetk

pip install Pillow
#or pip install -r teeko/requirements.txt

python3 teeko/main.py

```
